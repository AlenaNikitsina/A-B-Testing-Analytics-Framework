# A/B Testing Analytics Framework

Комплексный проект по анализу A/B-тестов, включающий валидацию системы сплитования (AA-тест), анализ результатов эксперимента (AB-тест) и применение продвинутых методов (линеаризация метрик). Все расчеты выполнены на реальных данных из ClickHouse с использованием статистических методов и визуализации.

## 🎯 Цели проекта

1. **Валидация системы сплитования:** Проверить корректность разделения пользователей на группы с помощью многократных AA-тестов.
2. **Анализ результатов AB-теста:** Оценить влияние нового алгоритма рекомендаций на метрику CTR, используя различные статистические методы.
3. **Применение линеаризации:** Повысить чувствительность теста при анализе метрик-отношений (лайки/просмотры) и сравнить результаты с обычным t-тестом.

## 📊 Набор данных

Данные получены из таблиц ClickHouse за период с `2025-11-14` по `2025-11-27`:

- **Таблица `feed_actions`:** Содержит информацию о действиях пользователей в ленте новостей:
    - `user_id` — идентификатор пользователя
    - `views` — количество просмотров
    - `likes` — количество лайков
    - `os` — операционная система
    - `gender` — пол
    - `age` — возраст
    - `date` — дата события
    - `group` — экспериментальная группа (0, 1, 2, 3)

## 🧪 Задание 1: Валидация системы сплитования (AA-тест)

**Цель:** Убедиться, что система сплитования работает корректно и не создает искусственных различий между группами.

### Методология:
1. Взят период AA-теста: `2025-11-14` по `2025-11-20`
2. Использованы группы **2** и **3**
3. Проведено **10 000 итераций**:
    - Из каждой группы случайно извлечены подвыборки по 500 пользователей (без возвращения)
    - Выполнены t-тест и Mann-Whitney U тест
    - Сохранены p-value
4. Построена гистограмма распределения p-value

### Результаты:
- Доля p-value < 0.05 составила **≈ 5%**
- Это соответствует ожидаемому уровню ложноположительных срабатываний при α = 0.05
- Распределение p-value равномерное (нет перекоса в сторону малых значений)

### Вывод:
✅ AA-тест пройден успешно. Система сплитования работает корректно, группы статистически неразличимы. Можно переходить к проведению AB-тестов.

## 📈 Задание 2: Анализ AB-теста (CTR)

**Цель:** Оценить эффект нового алгоритма рекомендаций во 2-й группе относительно контроля (1-я группа).

### Период эксперимента:
`2025-11-21` по `2025-11-27`

### Метрика:
CTR = `likes / views` (на уровне пользователя за весь период)

### Примененные методы:
1. **t-тест** (equal_var=False)
2. **Mann-Whitney U тест**
3. **Сглаженный CTR** (α = 5)
4. **Пуассоновский бутстреп**
5. **Бакетное преобразование** (бакеты по 1000 пользователей) + t-тест и Mann-Whitney

### Ключевые наблюдения:
- Распределение CTR во 2-й группе оказалось **бимодальным**, что нарушает предпосылки t-теста
- Все непараметрические методы (Mann-Whitney, бутстреп, бакеты) показали, что **CTR контрольной группы выше**
- Сглаживание не помогло нормализовать распределение из-за сильной бимодальности

### Вывод:
❌ **Новый алгоритм не рекомендуется к раскатке.** Несмотря на начальный рост в первые дни эксперимента, итоговый CTR оказался ниже контроля. Алгоритм требует доработки.

## 🔬 Задание 3: Линеаризация метрик

**Цель:** Применить метод линеаризации для повышения чувствительности теста при анализе метрики CTR.

### Суть метода:
Линеаризация преобразует метрику-отношение (лайки/просмотры) так, чтобы она сохраняла чувствительность теста, но при этом могла анализироваться обычным t-тестом.

### Сравниваемые группы:
- **Группы 0 (контроль) vs 3 (тест)**
- **Группы 1 (контроль) vs 2 (тест)**

### Результаты:

#### Группы 0 vs 3:
| Метод | p-value | Вывод |
|-------|---------|-------|
| Обычный CTR (t-тест) | 6.22e-44 | Различия значимы |
| Линеаризованные лайки | 1.49e-58 | ✅ p-value уменьшилось |

#### Группы 1 vs 2:
| Метод | p-value | Вывод |
|-------|---------|-------|
| Обычный CTR (t-тест) | 0.68 | ❌ Различия НЕ значимы |
| Линеаризованные лайки | 2.98e-09 | ✅ Различия стали значимыми |

### Выводы по линеаризации:
- Линеаризация **увеличила чувствительность t-теста** в обоих случаях
- Метод позволил обнаружить эффект там, где обычный t-тест его не видел (группы 1 vs 2)
- Линеаризация работает быстро и проверяет ту же гипотезу, что и пуассоновский бутстреп
- **Рекомендация:** Использовать линеаризацию как основной инструмент для анализа метрик-отношений в A/B-тестах
